# Локальный план ролей и связок

## 1. Поток ролей (базовый)
1. `Роддом/ЗАГС`: создается ребенок (ФИО ребенка + ФИО родителей).
2. `Школа`: по ребенку создается школьная запись.
3. `Университет`: после школы создается запись обучения в вузе/колледже.
4. `ВВК`: решение о годности.
5. `Военкомат`: начало службы / окончание службы.
6. `Граница`: временный статус "За границей" и возврат.

## 2. Статусы и переходы
- Школа:
  - `Учится` -> `Школьник`
  - `Закончил` -> `Свободен (ожидает)`
  - `Отчислен` -> `Другое (может быть указано в описании)`
- Университет:
  - `ENROLLED + FULL_TIME` -> `Учится в университете`
  - `ENROLLED + PART_TIME` -> `Свободен (ожидает)`
  - `EXPELLED` -> `Свободен (ожидает)`
- ВВК:
  - `DONE` -> `Годен к службе`
  - `IN_REVIEW` -> `Другое (может быть указано в описании)`
  - `WAITING` -> `Свободен (ожидает)`
- Граница:
  - `outsideBorder = true` -> `За границей`
  - `outsideBorder = false` и текущий статус `За границей` -> `Свободен (ожидает)`
- Военкомат (следующий шаг):
  - начало службы -> `На службе`
  - демобилизация -> `Служба окончена`
