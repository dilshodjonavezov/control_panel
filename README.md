# Система Военкомата

Проект предназначен для автоматизации работы военкомата и связанных ведомств: роддом, ЗАГС, ЖЭК, паспортный стол, школа, колледж/вуз, поликлиника, ВВК и пограничная служба.

Основная бизнес-идея:
- гражданин появляется в системе с рождения;
- его данные связываются с семьей, адресом и документами;
- военкомат автоматически получает основания для отсрочки или освобождения;
- жизненный путь гражданина должен быть виден в одной системе.

Подробное ТЗ находится в [docs/TZ-voenkomat.md](docs/TZ-voenkomat.md).

## Текущее состояние
- frontend: Angular
- backend: NestJS + MongoDB
- есть модули по рождению, ЗАГС, адресам, учебе, медицине, ВВК, границе и воинскому учету

## Запуск frontend
```bash
ng serve
```

## Запуск backend
```bash
cd backend
npm run start:dev
```

## Сборка
```bash
ng build
```
